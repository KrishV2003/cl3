1) sudo apt update
2) sudo apt install openjdk-8-jdk -y
3) java -version
----> sudo update-alternatives --config java
----> Type the number corresponding to Java 8.eg 3 and press Enter and then check java -version
----> cd ~  -> Go to Home Directory
4) wget https://downloads.apache.org/hadoop/common/hadoop-3.3.6/hadoop-3.3.6.tar.gz
5) tar -xvzf hadoop-3.3.6.tar.gz
6) sudo mv hadoop-3.3.6 /usr/local/hadoop
7) Configure Environment Variables:
   -- gedit ~/.bashrc
   -- add this lines below:
      -> export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64
         export HADOOP_HOME=/usr/local/hadoop
         export PATH=$PATH:$HADOOP_HOME/bin
         export PATH=$PATH:$HADOOP_HOME/sbin
   -- save file
   -- run: source ~/.bashrc
8) check: hadoop version
----------------------------
9) Configure Hadoop
   -- cd /usr/local/hadoop/etc/hadoop
   -- sudo gedit /usr/local/hadoop/etc/hadoop/core-site.xml
      --> Add inside <configuration>:

         <property>
             <name>fs.defaultFS</name>
             <value>hdfs://localhost:9000</value>
         </property>

       --> Save.
    -- sudo gedit /usr/local/hadoop/etc/hadoop/hdfs-site.xml
       --> Add:
          <configuration>

    		<property>
        		<name>dfs.replication</name>
        		<value>1</value>
    		</property>

	  </configuration>
       --> Save.
10) Format the NameNode : Run -> hdfs namenode -format	
11) Install and Enable SSH Server
    -- sudo apt install openssh-server -y
    -- sudo service ssh start
    -- sudo systemctl enable ssh
    -- sudo systemctl status ssh
    -- ssh localhost
    -- Type yes
    -- Generate SSH Key : ssh-keygen -t rsa -P ""
    -- Copy Key to Authorized Keys: cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
    -- chmod 600 ~/.ssh/authorized_keys
    -- Test again: ssh localhost
    -- Type: exit

12) Start Hadoop First Time:
    -- start-dfs.sh
    -- start-yarn.sh
    -- jps
    If restarting later:
    -- stop-dfs.sh
    -- stop-yarn.sh
    -- start-dfs.sh
    -- start-yarn.sh
    -- jps
13) Fix JAVA_HOME for Hadoop:
    -- readlink -f $(which java)
    -- Copy Parent Java Folder: /usr/lib/jvm/java-8-openjdk-amd64/jre/bin/java
    -- Remove /bin/java from end.
    -- sudo nano /usr/local/hadoop/etc/hadoop/hadoop-env.sh 
    -- (if necessary also edit: Ctrl + V)

       sudo nano /usr/local/hadoop/etc/hadoop/hdfs-env.sh

       sudo nano /usr/local/hadoop/etc/hadoop/yarn-env.sh

    -- Ctrl + W -> search JAVA_HOME and edit to export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64/jre
    -- Ctrl + O then press Enter then press Ctrl + X to exit
    -- Reload Bash: source ~/.bashrc
    -- Restart Hadoop: start-dfs.sh
                       start-yarn.sh
    -- jps: NameNode
	    DataNode
	    SecondaryNameNode
	    ResourceManager
            NodeManager
            Jps
14) Create Project Folder: 
    mkdir ~/weather
    cd ~/weather
15) Create Mapper File: nano WeatherMapper.java also we can use gedit instead of nano
16) Create Reducer File: nano WeatherReducer.java
17) Create Driver File: nano WeatherDriver.java
18) Compile Java Files: 
    javac -classpath `hadoop classpath` -d . WeatherMapper.java WeatherReducer.java WeatherDriver.java
19) Create JAR File: 
    jar -cvf weather.jar *.class
20) Create Input File:
    -- nano weather.txt
    -- Write:
       2018,32
       2018,35
       2018,30
       2019,40
       2019,42
       2019,39
       2020,25
       2020,27
       2020,26
       2021,45
       2021,43
       2021,44
    -- Save and exit.
21) Upload Input File to HDFS:
    -- hdfs dfs -mkdir /weatherinput
    -- hdfs dfs -put weather.txt /weatherinput
    -- hdfs dfs -ls /weatherinput
22) Run MapReduce Program:
    hadoop jar weather.jar WeatherDriver /weatherinput /weatheroutput
23) View Output:
    hdfs dfs -cat /weatheroutput/part-r-00000
24) Before rerunning:

    hdfs dfs -rm -r /weatheroutput

    Because Hadoop won’t overwrite output folders.

IF NAMENODE NOT APPEARS AFTER JPS:
STEP 1 — Stop Hadoop

Run:

stop-dfs.sh
stop-yarn.sh
STEP 2 — Check JAVA_HOME

Run:

readlink -f $(which java)

Example output:

/usr/lib/jvm/java-8-openjdk-amd64/jre/bin/java
STEP 3 — Edit hadoop-env.sh

Open:

sudo nano /usr/local/hadoop/etc/hadoop/hadoop-env.sh

Find:

JAVA_HOME

Replace with:

export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64/jre

IMPORTANT:

Remove /bin/java from end
STEP 4 — Also Edit These Files

Open:

sudo nano /usr/local/hadoop/etc/hadoop/hdfs-env.sh

and:

sudo nano /usr/local/hadoop/etc/hadoop/yarn-env.sh

Set same:

export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64/jre
SAVE NANO FILE

Press:

CTRL + O
ENTER
CTRL + X
STEP 5 — Reload Bash
source ~/.bashrc
STEP 6 — Reformat Namenode

IMPORTANT.

Run:

hdfs namenode -format

Wait until formatting completes successfully.

You should see:

Storage directory formatted

start-dfs.sh
start-yarn.sh

jps
